如何选择或安装客户端

1.选择合适您的客户端

我们优先推荐使用Minecraft Java Editction ，因为这样您可以使用服务器内的高级功能与模组，比如语音通话，小地图，物品信息等等

但如果您的设备无法流畅运行Minecraft Java Editction，我们则不推荐您使用它进行游戏，我们更推荐您使用占用更小的Minecraft Bedrock，它可以在您的电脑，手机，平板，游戏机等设备上运行，实现了随时随地

但由于服务器本身是Minecraft Java Editction，而实现Minecraft Bedrock兼容是使用geyser转译而成的，所以会存在一些无法解决的bug，并且不支持部分方块与特殊功能

2.安装

如您使用java端需要拥有一个正版账户，本服务器不支持离线

首先是电脑端，我们建议您使用完全开源的HMCL作为启动器，使用当前服务器版本和fabric作为模组加载器

其次是安卓手机，使用java版本，我们建议使用Fold Craft Launcher作为启动器，该启动器支持ipv6，并且基于HMCL-PE开发，并且至今仍在维护

接下来是基岩版，安卓端我们建议您前往mcapks.net下载最新正式版，不过有条件我们任然建议您支持正版，在google play购买

而IOS需要越狱，并且比较复杂，如果您的设备已经越狱并且配置好AppSync可以联系mozi获取安装包，如果您有条件支持正版，请在app store购买
