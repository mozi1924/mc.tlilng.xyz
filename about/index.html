服务器特点：

最新版本更新：

我们始终保持服务器使用最新版本的Minecraft，以确保玩家能够享受到最新的内容和功能。

跨平台支持：

无论您使用的是Minecraft Java Edition还是Minecraft Bedrock，您都可以加入我们的服务器，与朋友一同畅玩。

IPv6网络：

我们采用了IPv6网络，以提供更稳定、高速的连接，让您畅享游戏体验。

在线存档预览：

您可以在我们的网页界面中预览您的存档，随时查看您的世界的发展和变化。

纯净生存模式：

我们的服务器以纯原版生存模式为核心，让您能够在一个真实、原始的世界中展开您的冒险。

简洁游戏体验：

我们抛弃了花哨的特效和多余的元素，让您能够专注于探索、建设和与朋友互动。

强大的语音聊天(仅支持java版)：

我们使用Simple Voice Chat Mod作为语音聊天支持，您可以走到玩家身边与其聊天，就像在现实中聊天一样，您也可以创建一个群组，私密通话，解除距离和维度限制。

1.访问我们的网站，并点击导航栏上的预览选项

2.登录您的账户，即可开始预览

3.在网页上，您可以随时查看您的存档、与朋友分享您的成就，以及观察世界的变迁。
